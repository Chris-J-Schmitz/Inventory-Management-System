Inventory Management System

Allows a user to add, modify, or delete parts and products. 
Parts can either be in house or outsourced.
Products are composed of available parts in the system. 
Search function to allow user to search for parts and products based on their names or ID numbers. 

Program comes with filler parts and products already set up. These parts are set up as computer parts and computers. 
Though the user can set it up to be whatever part/products they choose. 

Created with JavaFX and Scene Builder.
